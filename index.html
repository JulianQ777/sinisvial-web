¿Cómo usar SinisVial?

1. Buscar dirección: Usa el campo de búsqueda para encontrar una ubicación.

2. Reportar un incidente: Haz clic en el botón "Reportar Incidente" y selecciona la ubicación en el mapa.

3. Buscar rutas seguras: Usa el mapa para encontrar rutas marcadas como seguras.

4. Emergencia: Presiona el botón "Emergencia" para contactar servicios de emergencia.
